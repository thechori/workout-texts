## Description

[Nest](https://github.com/nestjs/nest) API for the Workout Texts app

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

First, spin up local DB:

```bash
$ docker-compose up -d
```

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
